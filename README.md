# FlavoGen - AI Powered Recipe Sharing Platform

## Overview
**FlavoGen** is a full-stack recipe-sharing platform designed to bring together food enthusiasts and cooking experts in one interactive space. Built using the Django-React stack, it offers an intuitive and user-friendly interface for sharing, viewing, and managing recipes. The platform is enhanced with AI-powered chatbot integration, enabling users to quickly get recipe suggestions and culinary assistance. A secure login authentication system ensures that only registered users can access the platform.

---

## Features
- **User Authentication**:  
  - Secure login and registration system.  
  - Access restricted to registered users only.  

- **Recipe Management**:  
  - Share your favorite recipes with the community.  
  - Browse and view a wide collection of recipes curated by users.  
  - Add new recipes through a simple and intuitive interface.  

- **Engagement Features**:  
  - Rate and comment on your favorite recipes.  
  - Bookmark recipes for quick access in the future.  
  - Submit feedback through a dedicated form for continuous improvements.  

- **Search and Assistance**:  
  - Search for recipes using the built-in search functionality.  
  - AI-powered chatbot provides instant recipe suggestions and cooking tips.  

- **Admin Panel**:  
  - Manage user activity and platform content effectively.
